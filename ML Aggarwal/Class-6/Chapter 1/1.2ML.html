Question 1. Use the appropriate symbol < or > to fill in the blanks:

(i) 173 … 189
(ii) 1058 … 1074
(iii) 8315 … 8037

Answer:

(i) 173 < 189

(ii) 1058 < 1074

(iii) 8315 > 8037

Question -2. In each of the following pairs of numbers, state which number is smaller:

(i) 553,503
(ii) 41338,1139
(iii) 25431,24531

Answer:

(i) Smallest Number: 503

(ii) 1139 is smallest number.

(iii) 24531 is smallest number.

Question 3. Find the greatest and the smallest numbers in each row:

(i) 71834, 75284, 571, 2333, 594
(ii) 9853, 7691, 9999, 12002.

Answer:

(i) Smallest number: 571

Greatest number: 75284

(ii) Smallest number: 7691

Greatest number: 12002

Question -4. Arrange the following numbers in ascending order:

304, 340, 34, 43, 430

Answer:

34 < 43 < 304 < 340 < 430.

Question- 5. Arrange the following numbers in descending order:

53, 7333, 553, 7529, 335.

Answer:

7529 > 7333 > 553 > 335 > 53

Question- 6. Arrange the following numbers in ascending order:

67201, 743819, 748391, 67218, 85344

Answer:

67201 < 67218 < 85344 < 743819 < 748391

Question- 7. Arrange the following numbers in descending order:

36957, 37140, 723891, 732142, 5718901, 37401

Answer:
